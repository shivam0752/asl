Place your custom app icons/logos in this folder.

Suggested filenames:
- tab-home.png
- tab-character.png
- tab-progress.png
- tab-profile.png
- linkedin-logo.png
- googlefit-logo.png
- class-reveal-avatar.png

Then wire them in `constants/assets.ts`, for example:

`tabHome: require('../assets/branding/tab-home.png')`

Notes:
- Prefer transparent PNGs.
- Keep tab icons around 24x24.
- Keep provider logos around 40x40.
